build with taro, vant-weapp

关键词 Taro vant-weapp

在开发小程序中，尝试使用Taro作为基础框架，引入vant-weapp，Taro-UI。 引入后者相对来说要简单一些， 但是引入前者比较麻烦，我引入成功后分享出来， 希望帮助到更多的朋友，降低整合成本。

  1. Taro ^1.1.0-beta.2 或 ^1.1.0
  2. Type Script

整合有赞的组件库 vant-weapp。 可以做整合的示例。

Taro 官网地址: https://github.com/NervJS/taro

Vant-weapp文档地址: https://youzan.github.io/vant-weapp

安装Taro的新版本

  1. 使用taro init myApp
  2. 修改package.json中的taro相关包的版本号*^1.1.0-beta.2
  3. 运行npm install

整合要点

  1. 下载vant-weapp/dist保存在src/components/vant-weapp/dist
  2. 在首页index.jsx中修改usingComponents,声明引入的组件
  3. 在render方法中直接组件比如van-button

反清单管理，人生不只是效率

在我们的职业生涯过程中，有很多的书教育我们要讲清单管理，要讲究时间管理。今天我就讲一讲反清单管理，就是人生不只是效率。

或者说我们每次谈到清单管理的时候，对于一个小事情来讲，是追求此时此刻的效率。如果放在更长的一个时间轴上，我们追求的是在这一个相当长的时间轴上的时间效率。

我们既要追求当下的效率，也要追求更加长久的效率，更要关注我们自身。

在合适的时间创造

发现一个现象，就是如果说我今天晚上睡得好的话，明天早晨到办公室之后，愿意从事一些创造性的工作。如果今天晚上睡得不好，明天早晨去上班是可以的，打打酱油可以，但是创造性的工作一点儿都不愿意去干，也不愿意启动。

中场休息

读书期间，有的老师啊，就是下课之后呢，就抽烟，在讲台上抽烟，为什么要抽烟呢？他要缓解自己的疲劳，还有一些老师呢，在课间的时候明确跟我们讲，我下课之后你们不要问我问题啊，就让我自己呆着。

当时不解老师为啥子。

发现自己的阈值

我后来也是发现身体有一个阈值，达到这个阈值之后呢，好像这个开关突然就关闭了，我在加班也不会干活，后来想想算了，干脆只要达到这个阈值，我干脆就出去遛一遛。

漫漫的人生路上，我们既要对职场负责，也要对我们自身负责，如果人到中年的话，我们又承担着上有老下有小的责任。希望我们强调清单管理，强调时间管理的时候，偶尔也可以适当的放纵一下自己，偶尔的反清单，没有什么大碍。偶尔的反清单，我们依旧可以拥有更加美好的人生。

学习曲线第一个拐点

第一个拐点

前一段时间重新识别学习曲线。找到了学习曲线的两个拐点，我把第一个拐点定义成你学习了10~%30，这个时候你头痛欲裂也非常恼火。

再坚持下去或者是实际动手操作一个完整的案例，可能就变成了熟手。

但是如果在第一个拐点选择了放弃暂停，比如暂停一两周。学习成果很快会回到起点（S），就是第二张图的zero。

这个背后是由神经科学的依据，在第一个拐点，你表现出来的痛苦啊，头痛欲裂甚至焦躁，都是在大脑里面产生出来很多树突，轴突。但是还没有连接成网状。

接纳自己在第一个拐点的笨拙。

特别是一些聪明人，经常会在第一个拐点的时候，质疑不适合于学习这个门类。选择放弃不是因为懒惰，而是因为质疑自己。

既然第一个拐点的笨拙，痛苦都是正常的，有神经科学的依据，那就可以坦然接纳自己在第一个拐点的表现。然后坚持下去。

第二个拐点

第一个拐点以后突然加速，第二个拐点后开始平滑。没闯过第一个拐点，就快速退回到零点。过几周几个月，又是从零点开始。

在职场中，很多年轻人就在这两个拐点上吃大亏。

在第一个拐点选择放弃，在第二个拐点选择放纵。

适合新码农的改善伙食解决方案

新码农的特点，刚刚参加工作，没有女朋友，租房居住，居住面积不大，不适合起明火。

日常吃饭，除了在办公室吃食堂外呢，就是点外卖。

即使非常卫生的外卖餐食，也不可避免了多加盐和味精，长期使用不利于身心。

下面总结适合新码农的改善伙食解决方案。

基础工作

买碗碟筷子，适合两三人就餐的餐具，洗洁精，洗碗布，小案板，菜刀。

小案板买塑料材质的，不要买木头材质的。

我们做不到每日开火做饭，木头案板多日不用，容易开裂

小米粥

买一个电饭煲，推荐可以做柴火饭的那一款，可以定时，可以预约。

买一斤装的小米。

早晨出门前，淘米，加水，设置预约时间，晚上回到家就可以喝了。

即使加班迟了，到家喝点小米粥也无妨，只是小米粥更黏糊了。

萝卜排骨

买一个压力锅，使用上面提到的电饭煲也可以。

适合周末做。 买一根萝卜，两斤排骨。有些同学吃萝卜烧心，也可以切成段的冬瓜。

买些花椒，茴香，大料，辣椒，十三香等炖料。

买两三根黄瓜，大蒜，酱油醋，香油，芥末油，准备做凉拌黄瓜，解油腻。

周六上午11点去附近超市采购，回来做，中午正巧吃。主食焖米饭或买满头都可以。

涮羊肉或火锅

买一个电磁炉，一口适合电磁炉的锅，锅不要太大，适合两三人聚餐即可。

买两三盒羊肉片，买些青菜，毛肚，鸭肠，猪血也可以买一些回家来做。

适合周末三五好友聚餐。

上述用到的厨房餐具炊具切莫一次性置办齐整，慢慢来即可。

突破原点盘旋

两人间的唯一区别，你有更多选择，而我只有一个选择。

从更多选择出发，首先踌躇，犹豫不决，迟迟不能转换成为行动。

从唯一选择出发。看到底是否可行，怎么执行，然后开始第一步，做下去直到彼岸。

每一个优秀的人，都不是与生俱来带着光环的，也不一定是比别人幸运。他们只是在任何一件小事上，都对自己有所要求，不因舒适而散漫放纵，不因辛苦而放弃追求。雕塑自己的过程，必定伴随着疼痛与辛苦，可那一锤一凿的自我敲打，终究能让我们收获一个更好的自己。

见过太多原点盘旋，在原点耗费太多时间精力确认最佳方案。什么是最佳，可能需要在实践中逐渐验证。

在原点盘旋阶段，也有效率要素。有效率地确认当下的唯一选择，不能确认的时候抛硬币选择其一，开始行动，胜过原地打转转琢磨哪一个选择最棒。
