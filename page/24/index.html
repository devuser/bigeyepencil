  1. macOS
  2. Docker
  3. Node.js v8.12
  4. Taro

这个开发环境消耗比较多的硬件资源，如果硬件资源受限的同学，鼓励使用本机环境，不要使用Docker环境。

  1. 安装 Node.js，访问如下网址，选择适合您开发机的版本

  2. 升级 npm

  3. 安装 Taro

    在Shell窗口中输入 taro help确认Taro的版本号。
